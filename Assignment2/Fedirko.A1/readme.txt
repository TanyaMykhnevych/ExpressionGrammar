There are two projects in current directory 'Expressions' and 'Expressions-Main'
Expressions contains antlr files: .g4 file and etc
Expressions-Main contains project with simple java console app, which able to run grammar rules
 through test files located in ./Expressions/input

Its possible to run in two approaches:

Open whole directory in eclipse as workspace
    1. Use text from files in ./Expressions/input directory and insert into a Parse tree window

    or

    2. Or simply run 'Expressions-Main' project with 'ExpressionsMain' run configuration
